The cycling chain powered by you

Designed for durability

The lock will be designed to be Burglar proof and easy to use, so that you can rest assured your bike is safe while parked.

IoT development to ensure privacy and safety

Bikera will be privacy minded building the firmware and electrical design. This will make sure you are in control of the information about your location and will only be shared with the network on your request low powered MCU's with Bluetooth and LoRa capabilities will be used so that you don't have to charge the lock often.

Validator nodes communicate privately

Using secure keys to interact with the IoT lock; data output will be dynamic cryptografically secured. Interacting with the validators will be end-to-end encrypted with rotating/rolling security keys. For secure radio communication.

The blockchain only registers, miles and minutes driven

The blockchain will link the IoT lock through LoRa gateways. Proof-Of-Transit will be using the difference in measured time elapsed between first en second data transfer radio signal. This difference can be used to calculate distance traveled between radio signals. Using only 2 gateway nodes creates a vector of a certain timeframe but has an optional second false vector; speed and distance traveled can be verified without giving away your location. you need begin and endpoint to know which vector is correct.

Lock is available as open hardware initiative

The open source, open hardware public available repo's and open development will make sure Bikera will be available for everyone everybody who wants to participate in production or development is encouraged to do so. Bug hunting bounties will be placed as initiatives to incentevise collaboration, this in all aspects of Bikera; mechanical, electrical, IoT development, Blockchain development

Games will be easily creatable

Geo location games will be added to bikera Blockchain creating versions of games like a race will be easely done in the app. Location can be allowed to be used to set land markers, these in turn can be used in games like capture the flag. Creating your own custom map or version
